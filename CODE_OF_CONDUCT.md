# Code of Conduct

## Scope

This code of conduct applies to all project spaces including issues, pull requests, and any discussions related to Super Mod Checker.

## Standards

Acceptable behavior:

- Constructive feedback focused on the code and the project.
- Clear, factual bug reports with logs and reproduction steps.
- Respecting that this is a mod detection tool for Gorilla Tag, not a general modding platform.

Unacceptable behavior:

- Harassment, personal attacks, or discriminatory language.
- Requests to add features that assist cheating or harm other players.
- Sharing or distributing exploits, cheat clients, or tools designed to evade mod detection.

## Enforcement

Issues or pull requests that violate these standards will be closed without discussion. Repeat offenders will be blocked from the repository.
